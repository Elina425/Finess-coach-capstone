# Fitness coach capstone (code)

Python pipeline for pose estimation, keypoint preprocessing, biomechanical features, and BiLSTM / GCN training on the QEVD-FIT-COACH-style layout.

**Documentation:** start with [`docs/CAPSTONE_PIPELINE.md`](docs/CAPSTONE_PIPELINE.md) (code and data map, command order) and [`docs/README.md`](docs/README.md) (index of capstone docs).

**Environment:** `python -m venv venv && ./venv/bin/pip install -r requirements.txt`

**Common entry points:** `keypoint_preprocessing_pipeline.py`, `train_exercise_bilstm.py`, `run_full_comparison.py`, `quickstart.py`, `run_pipeline.py`.
